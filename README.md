## Install

NodeJS

```
node -v
```

Typescript and Modules

```
npm i typescript -D
```

Express
```
npm i express
npm i @types/express -D
npm i express-validator
```

Mongo
```
npm i mongoose
npm i @types/mongoose -D
```

CORS
```
npm i cors
```


## Config

NodeJS

```
npm init -Y
vi package.json
```

Typescript

```
tsc --init
vi tsconfig.json
```

## Run

```
tsc
node build/index.js
```
